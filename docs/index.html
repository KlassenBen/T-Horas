¿Que me ofrece T Horas?

Esta aplicación te ahora el tiempo de cada sábado sumar las horas y calcular el sueldo de tus trabajadores.

  • Puedes apuntar las horas de cada trabajador aqui.
  • Las horas son guardados en la nube, y los puedes acceder de cualquier dispositivo.
  • Tus trabajadores pueden vincularse con tu equipo y ver sus propias en su propio celular.

Toca aqui para ver como instalar esta aplicación

  • En tu navagador, busca la opción de añdirlo a la pantalla principal.
  • Para IOS (iPhone), toca el icono de compartir, luego añadir a la pantalla principal.
  • Para OS (Android), toca el icono ⠇, luego instalar aplicación, o añadir a la pantalla principal.
